# ConsumerRelationshipManagement
Fully working simple CRUD app with Spring MVC and Hibernate.

Application allows to list, create, edit and delete customers.
## Technologies:
* Java 10
* Spring 5
* Spring MVC
* Hibernate 5
* PostgreSQL 10
